# Freerdp_Gui
FreeRDP GUI Wrapper

RDP bağlantılarını kolayca yönetmek ve kurmak için geliştirilmiş, FreeRDP tabanlı basit bir grafik arayüz (GUI) uygulamasıdır. (Uzak Masaüstü Ağ Geçidi - RD Gateway desteği mevcuttur). 
Özellikler

    Sezgisel Arayüz: Uzak bağlantıları kurmak ve yönetmek için grafik arayüz. (Geliştirilme aşamasında...)

    Dinamik Çözünürlük: Pencere boyutuna göre çözünürlüğü otomatik ayarlama. (Çalışıyor!)

    Sertifika Doğrulama: Güvenlik seçenekleri. (Bir FreeRDP hatası nedeniyle her zaman açık olmalıdır.)

    Pano Paylaşımı: Metin kopyala-yapıştır desteği. (Test edilmedi.)

    Özel Çözünürlük: Manuel çözünürlük ayarları. (Test edilmedi.)

    Tam Ekran Modu: Kesintisiz tam ekran deneyimi. (Çalışıyor!)

    Ses Yönlendirme: Uzak bilgisayardaki sesleri yerel makineye aktarma. (Henüz çalışmıyor :( )

    Yerel Sürücü Yönlendirme: Yerel klasörleri uzak masaüstüne bağlama. (Test edilmedi.)

    Yazıcı Yönlendirme: Yerel yazıcıyı uzak oturumda kullanma. (Test edilmedi.)

    Çoklu Monitör Desteği: Birden fazla ekran kullanımı. (Çalışıyor!)

    RD Gateway Desteği: Uzak Masaüstü Ağ Geçidi entegrasyonu. (Çalışıyor!)

    Yapılandırma Kaydı: Ayarları kaydetme ve yeniden yükleme imkanı. (Çalışıyor!)
